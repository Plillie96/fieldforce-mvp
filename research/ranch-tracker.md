# Ranch Candidate Tracker

Live pipeline of candidate properties. Score each against the scorecard in
[`docs/02-ranch-search-criteria.md`](../docs/02-ranch-search-criteria.md).
Only properties that clear the hard requirements *and* beat the weighted-score
threshold advance to a site visit.

## Pipeline

| # | Property / location | Acres | Ask price | $/deeded AU* | Water rights | USDA plant distance | Weighted score | Stage | Notes |
|---|---------------------|-------|-----------|--------------|--------------|---------------------|----------------|-------|-------|
| _ex_ | (example) NE Wyoming | 2,400 | $2.4M | ~$14k/AU | Deeded wells + creek | ~2.5 hr | — | Sourcing | Verify stocking rate w/ NRCS |
|   |                     |       |           |              |              |                     |                |       |       |

\* Cost per **animal unit of sustainable carrying capacity** — the number that
actually matters, not $/acre (see doc 02).

## Stages
`Sourcing → Scored → Site visit → Offer → Due diligence → Closed / Passed`
