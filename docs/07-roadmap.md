# 07 — Roadmap & Critical Path

The venture has one hard dependency chain and several things that can (and
should) run in parallel. The insight that shapes the whole timeline: **you can
validate the beef brand before you own a single acre**, and doing so de-risks
everything downstream — including the loan application.

## The critical path

```
Define criteria ──► Source & score ranches ──► Get financing pre-qual
        │                                              │
        └──────────────► Prototype DTC demand ─────────┘
                         (partner-sourced beef)         │
                                                        ▼
                              Offer ─► Due diligence ─► Close ─► Stock herd
                                                                    │
                                                                    ▼
                                            Own-supply DTC ─► Scale subscription
```

## Phase 3 — Operate & transition supply
- [ ] Take possession; repair/upgrade priority infrastructure
- [ ] Stock the herd
- [ ] Lock ongoing USDA processing slots
- [ ] Transition DTC supply from partner-sourced to own herd as cattle finish

## Phase 4 — Scale
- [ ] Grow subscribers to match herd throughput
- [ ] Expand SKUs (nose-to-tail, premium, bulk shares)
- [ ] Add complementary income (agritourism, conservation programs)
- [ ] Evaluate second supply source / owned processing only when demand exceeds
      one ranch's output

## The three things to get right, in order
1. **Don't buy the wrong land.** It's the one irreversible decision (doc 02).
2. **Structure the deal so the DTC margin services the debt** — not cattle
   income alone (docs 03, 05).
3. **Prove people will buy the beef before you're on the hook** for the mortgage
   and the herd (doc 04, Track C).
