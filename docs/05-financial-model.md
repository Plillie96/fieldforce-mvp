# 05 — Financial Model

> **Every number below is an illustrative assumption**, flagged so you can
> replace it with real quotes. The point of this doc is the *structure* of the
> economics and which levers matter — not precision. Update the parameter table
> in the [README](../README.md) and the assumptions here as real data arrives.

## The one equation that matters

**Value captured per carcass = (DTC revenue from one animal) − (all-in cost of
that animal, delivered).**

If that number, multiplied by throughput, covers land carrying cost + overhead
and leaves profit, the venture works. Everything else is detail.

## Illustrative per-animal economics (grass-finished steer)

| Line | Assumption | Notes |
|------|-----------|-------|
| Live weight at finish | ~1,150 lb | Grass-finished, ~24–28 months |
| Hanging (carcass) weight | ~630 lb | ~55% of live |
| Packaged, sellable weight | ~440 lb | ~65–70% of carcass after cutting/loss |
| **Cost side** | | |
| Cost to raise (or buy) to finish | $1,400 | Feeder + pasture/forage + vet/mineral |
| Processing (USDA, cut & wrap) | $850 | Slaughter fee + per-lb cut & wrap |
| Packaging & freezing | $150 | |
| **All-in cost per animal** | **~$2,400** | ≈ **$5.45 / sellable lb** |
| **Revenue side (DTC)** | | |
| Blended DTC price | $11.00 / lb | Weighted across ground→premium cuts |
| **DTC revenue per animal** | **~$4,840** | 440 lb × $11.00 |
| Shipping (if shipped, not bulk-share) | −$700 | Frozen 2-day; lower for bulk/local pickup |
| **Contribution per animal** | **~$1,740** | Before customer acquisition & overhead |

Commodity-market comparison: that same steer sold at auction might fetch
**~$1,500–1,800 total**. Selling it DTC grosses **~$4,840** — roughly **2.7×**.
That multiple *is the business*. The DTC premium is what services the land debt
that cattle income alone cannot.

## Box-level unit economics (subscription)

| Line | Assumption |
|------|-----------|
| Box retail price | $159 |
| Product COGS (allocated) | −$55 |
| Packaging + dry ice | −$12 |
| Shipping | −$28 |
| Payment processing (~3%) | −$5 |
| **Contribution margin per box** | **~$59 (37%)** |
| Blended CAC (target) | < $60 |
| Target repeat / retention | Payback in ≤ 2 boxes |

Levers, in order of impact: **shipping cost**, **whole-carcass sell-through**
(don't discount ground to clear it), **retention**, then CAC.

## Startup capital allocation (against assumed $2.0M)

| Bucket | Illustrative | Rationale |
|--------|-------------|-----------|
| Land down payment | $1.1M | ≤ ~55–60% of capital (per doc 03 rule) |
| Herd / initial cattle | $250k | Foundation cows or feeders to finish |
| Infrastructure & equipment | $150k | Fencing repair, handling, freezers, hauling |
| Brand / e-commerce / packaging | $120k | Site, design, first inventory, launch marketing |
| Working capital & runway (12–18 mo) | $380k | The buffer that keeps a bad year from ending it |
| **Total** | **$2.0M** | |

> **The failure mode to avoid:** pouring nearly all capital into the down
> payment and starting with no operating runway. Ranches have bad years
> (drought, cattle prices, a slow launch). Runway is survival.

## Ongoing annual carrying cost to cover (illustrative)

- Debt service on the financed land
- Property taxes & insurance
- Feed/hay in winter, vet, fuel, repairs
- Processing (variable with volume)
- Brand opex: platform fees, marketing, fulfillment labor

The DTC contribution must cover all of the above **plus** profit. Model it
monthly once real land price, herd size, and processor quotes are in hand.
